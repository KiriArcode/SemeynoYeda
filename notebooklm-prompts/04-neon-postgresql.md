# Промпт: Neon PostgreSQL

## Контекст проекта

Я работаю над **SemeynoYeda** (https://semeyno-yeda.vercel.app/) — PWA-приложением. Основная база данных — Neon PostgreSQL, доступ через Vercel Serverless Functions API. Нужны материалы по работе с Neon.

## Задача

Найти материалы по **Neon PostgreSQL**:

- Neon Serverless Postgres архитектура
- Работа с Neon через @neondatabase/serverless
- Миграции схемы данных в Neon
- Оптимизация запросов к Neon
- Connection pooling и управление соединениями
- Интеграция Neon с Vercel Serverless Functions
- Best practices для serverless PostgreSQL

## Формат результатов

1. **Краткое описание** — что это и зачем нужно
2. **Ключевые концепции** — основные идеи и принципы
3. **Best practices** — рекомендации по использованию
4. **Примеры кода** — практические примеры
5. **Ресурсы** — документация, статьи, видео
