# Промпт: Data Flow паттерны

## Контекст проекта

Я работаю над **SemeynoYeda** — PWA-приложением с сложной структурой данных (рецепты, меню, заготовки). Нужны материалы по управлению потоком данных.

## Задача

Найти материалы по **Data Flow паттернам**:

- Unidirectional data flow
- Подъём состояния (lifting state up)
- Централизованное управление данными
- Оптимизация запросов к данным
- Кэширование данных на клиенте

## Формат результатов

1. **Краткое описание** — что это и зачем нужно
2. **Ключевые концепции** — основные идеи и принципы
3. **Best practices** — рекомендации по использованию
4. **Примеры кода** — практические примеры
5. **Ресурсы** — документация, статьи, видео
6. **Связь с проектом** — применение в SemeynoYeda
