# Промпт: Оптимизация PWA

## Контекст проекта

Я работаю над **SemeynoYeda** — PWA-приложением. Нужны материалы по оптимизации производительности PWA.

## Задача

Найти материалы по **Оптимизации PWA**:

- Стратегии кэширования Service Worker
- Оптимизация размера бандла
- Preloading критических ресурсов
- Оптимизация загрузки шрифтов
- Оптимизация работы с IndexedDB

## Формат результатов
