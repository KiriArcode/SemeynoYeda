# Промпт: CI/CD

## Контекст проекта

Я работаю над **SemeynoYeda** — PWA-приложением. Нужны материалы по автоматизации сборки и деплоя.

## Задача

Найти материалы по **CI/CD**:

- Автоматизация сборки и деплоя
- Тестирование перед деплоем
- Версионирование релизов
- Rollback стратегии
- Мониторинг деплоев

## Формат результатов

1. **Краткое описание** — что это и зачем нужно
2. **Ключевые концепции** — основные идеи и принципы
3. **Best practices** — рекомендации по использованию
4. **Примеры кода** — практические примеры
5. **Ресурсы** — документация, статьи, видео
6. **Связь с проектом** — применение в SemeynoYeda
