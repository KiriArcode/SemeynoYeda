# Промпт: Управление рецептами

## Контекст проекта

Я работаю над **SemeynoYeda** — приложением для управления рецептами. Нужны материалы по структурированию и работе с данными рецептов.

## Задача

Найти материалы по **Управлению рецептами**:

- Структурирование данных рецептов
- Парсинг рецептов из текста
- Хранение и поиск рецептов
- Версионирование рецептов
- Импорт/экспорт рецептов

## Формат результатов

1. **Краткое описание** — что это и зачем нужно
2. **Ключевые концепции** — основные идеи и принципы
3. **Best practices** — рекомендации по использованию
4. **Примеры кода** — практические примеры
5. **Ресурсы** — документация, статьи, видео
6. **Связь с проектом** — применение в SemeynoYeda

## Инструкции

- Фокус на практических материалах (2023-2026)
- Русский и английский языки
- Указывай сложность (beginner/intermediate/advanced)
- Приоритет: низкий
