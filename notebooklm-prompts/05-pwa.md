# Промпт: PWA (Progressive Web Apps)

## Контекст проекта

Я работаю над **SemeynoYeda** (https://semeyno-yeda.vercel.app/) — PWA-приложением для управления семейным питанием, деплоится на Vercel. Нужны материалы по Service Worker, offline функциональности и оптимизации.

## Задача

Найти материалы по **PWA (Progressive Web Apps)**:

- Service Worker стратегии кэширования
- Offline-first подход к разработке
- Манифест PWA и установка на устройство
- Background sync и push-уведомления
- Оптимизация производительности PWA
- Работа с файлами и медиа в PWA

## Формат результатов

1. **Краткое описание** — что это и зачем нужно
2. **Ключевые концепции** — основные идеи и принципы
3. **Best practices** — рекомендации по использованию
4. **Примеры кода** — практические примеры
5. **Ресурсы** — документация, статьи, видео
6. **Связь с проектом** — применение в SemeynoYeda
