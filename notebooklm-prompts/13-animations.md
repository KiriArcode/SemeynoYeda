# Промпт: Анимации и переходы

## Контекст проекта

Я работаю над **SemeynoYeda** — PWA-приложением с плавными анимациями. Нужны материалы по оптимизации и accessibility анимаций.

## Задача

Найти материалы по **Анимациям и переходам**:

- Плавные анимации в веб-приложениях
- CSS transitions и animations
- Производительность анимаций
- Accessibility анимаций (prefers-reduced-motion)
- Микроинтеракции для улучшения UX

## Формат результатов

1. **Краткое описание** — что это и зачем нужно
2. **Ключевые концепции** — основные идеи и принципы
