# Промпт: Компонентная архитектура

## Контекст проекта

Я работаю над **SemeynoYeda** — PWA-приложением на React. Нужны материалы по организации компонентов и управлению состоянием в больших приложениях.

## Задача

Найти материалы по **Компонентной архитектуре**:

- Организация компонентов в больших приложениях
- Композиция vs наследование
- Управление состоянием (Context API, Zustand, Redux)
- Переиспользование компонентов
- Тестирование React компонентов

## Формат результатов

1. **Краткое описание** — что это и зачем нужно
2. **Ключевые концепции** — основные идеи и принципы
3. **Best practices** — рекомендации по использованию
4. **Примеры кода** — практические примеры
5. **Ресурсы** — документация, статьи, видео
6. **Связь с проектом** — применение в SemeynoYeda

## Инструкции

- Фокус на практических материалах (2023-2026)
- Русский и английский языки
- Указывай сложность (beginner/intermediate/advanced)
- Приоритет: средний
