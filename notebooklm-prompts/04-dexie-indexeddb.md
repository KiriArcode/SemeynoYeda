# Промпт: Dexie.js (IndexedDB)

## Контекст проекта

Я работаю над **SemeynoYeda** (https://semeyno-yeda.vercel.app/) — PWA-приложением с offline-first подходом. Использую Dexie.js для работы с IndexedDB как локальным кэшем и синхронизацией с основной БД Neon PostgreSQL через API. Нужны материалы по архитектуре и оптимизации.

## Задача

Найти материалы по **Dexie.js (IndexedDB)**:

- IndexedDB как локальный кэш для синхронизации с серверной БД
- Работа с IndexedDB через Dexie
- Миграции схемы данных
- Оптимизация запросов к IndexedDB
- Синхронизация IndexedDB ↔ Neon PostgreSQL через API
- Best practices для хранения больших объёмов данных в браузере
