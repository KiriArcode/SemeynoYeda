# Промпт: Supabase для фронтенда

## Контекст проекта

Я работаю над **SemeynoYeda** (https://semeyno-yeda.vercel.app/) — PWA-приложением. Supabase используется только для фронтенда (не для БД, основная БД — Neon). Нужны материалы по использованию Supabase на клиенте.

## Задача

Найти материалы по **Supabase для фронтенда**:

- Supabase Client для React приложений
- Supabase Auth на клиенте
- Supabase Realtime для синхронизации
- Supabase Storage для файлов
- Интеграция Supabase с PWA
- Best practices для клиентского использования Supabase

## Формат результатов

1. **Краткое описание** — что это и зачем нужно
2. **Ключевые концепции** — основные идеи и принципы
3. **Best practices** — рекомендации по использованию
4. **Примеры кода** — практические примеры
5. **Ресурсы** — документация, статьи, видео
6. **Связь с проектом** — применение в SemeynoYeda
